# 清梦 - 英雄联盟助手 Beta 0.1

一个基于 LCU API 的简易英雄联盟助手，提供对局自动监测、队友/敌方战绩查询以及图形化界面。

##  已实现功能

- ✅ 账号实时监测（显示召唤师信息）
- ✅ 对局阵容自动识别（选人阶段实时更新）
- ✅ 双方近期战绩查询（胜率、KDA、综合评分）
- ✅ 简易图形化界面（目前基于 tkinter）

##  已知 Bug

- 训练营在选人阶段退出会导致阶段判断异常（待修复）
- 部分娱乐模式的模式名称显示异常（不影响使用）

##  后续目标

- 设计更美观的图形化界面
- 优化代码结构与性能
- 增加主动输入召唤师名+编号的战绩查询功能
- 修复上述已知 Bug

## 📦 安装与使用

### 环境要求
- Windows 操作系统（需已安装并登录英雄联盟客户端）
- Python 3.8+（如需从源码运行）

### 从源码运行

1.克隆本仓库

   ```bash
   git clone https://github.com/你的用户名/你的仓库名.git
   cd 你的仓库名
   ```

2.安装依赖
   ```bash
   pip install lcu-driver requests
   ```

3.运行
   ```bash
   python gui.py
   ```
