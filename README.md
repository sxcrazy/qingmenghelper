# 清梦 - 英雄联盟助手 Beta 0.3

一个基于 LCU API 的简易英雄联盟助手，提供对局自动监测、队友/敌方战绩查询以及图形化界面。

##  已实现功能

- ✅ 客户端状态监测
- ✅ 对局自动确认
- ✅ 账号实时监测（显示召唤师信息）
- ✅ 对局阵容自动识别（选人阶段实时更新）
- ✅ 双方近期战绩查询（胜率、KDA、综合评分）
- ✅ 主动查询玩家战绩（召唤师名字#1234）
- ✅ 简易美观的图形化界面（目前基于 PySide6 ）

## 📦 安装与使用

### 环境要求
- Windows 操作系统（需已安装并登录英雄联盟客户端）
- Python 3.8+（如需从源码运行）

### 从源码运行

1.克隆本仓库

   ```bash
   git clone https://github.com/你的用户名/你的仓库名.git
   cd 你的仓库名
   ```

2.安装依赖
   ```bash
   pip install lcu-driver requests
   ```

3.运行
   ```bash
   python gui.py
   ```

### 简易食用方法：

· 从Releases下载清梦助手.zip

· 解压并打开.exe文件

· 启动游戏和程序
